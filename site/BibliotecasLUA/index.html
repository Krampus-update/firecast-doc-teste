Bibliotecas LUA

Este tópico contém a referência das bibliotecas e funções disponíveis para serem usadas em códigos LUA

 

 

Biblioteca Descrição
NDB Biblioteca para manipular NodeDatabases, documentos de fácil escrita e leitura para se armazenar informações  Leia mais em: Biblioteca ndb. 
VHD Biblioteca do HD Virtual do plug-in, usado para criar, ler e manipular arquivos.  Leia mais em: Biblioteca vhd. 
Dialogs Biblioteca que contém funções para exibir mensagens e interagir com o usuário através de diálogos.  Leia mais em: Biblioteca dialogs. 
GUI Biblioteca que contém funções relacionadas à interface e Lua Forms.  Leia mais em: Biblioteca gui. 
Locale Biblioteca que contém funções para lidar com idiomas e regiões, incluindo funcionalidade para permitir um plug-in em várias línguas.  Leia mais em Biblioteca Locale 
Firecast Biblioteca que contém funções relacionadas às mesas, personagens, envio de mensagens, rolagem de dados, etc..  Leia mais em: Biblioteca Firecast. 
Firecast.Messaging Biblioteca que contém funções que tratam as notificações que o Firecast dispara para todos os plug-ins através de mensagens.  Leia mais em Biblioteca Firecast.Messaging 
Firecast.Plugins Biblioteca que contém funções relacionadas aos plug-ins instalados no RRPG.  Leia mais em: Biblioteca Firecast.Plugins 
Firecast.Requests Biblioteca que contém funções que solicitam determinadas ações ao Firecast, como, por exemplo, criar itens na biblioteca da mesa, obter informações de perfil de um usuário, etc..  Leia mais em Biblioteca Firecast.Requests 
Internet Biblioteca que contém funções relacionadas à internet/intranet, como: downloads, requisições HTTP, etc..  Leia mais em Biblioteca internet 
FireDrive Biblioteca para manipular o FireDrive do usuário.  Leia mais em Biblioteca fireDrive 
SceneLib Biblioteca com funções para lidar com Scene 3 do RRPG.  Leia mais em Biblioteca scene 
System Biblioteca com funções para lidar com o dispositivo do usuário e outras funcionalidades básicas.  Leia mais em Biblioteca system 
Utils  Biblioteca que possui funções utilitárias diversas  Leia mais em Biblioteca utils.
