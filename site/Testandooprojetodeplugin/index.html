Testando o projeto de plug-in

    Para testar seu projeto de plug-in, ele deve ser compilado e instalado no RRPG. Siga estes passos:

  1. Abra o prompt de comando.
  2. Navegue pelo prompt até a pasta do projeto de plug-in.
  3. Execute o comando “rdk i”, sem as aspas, dentro da pasta para compilar e instalar o projeto de plug-in.
  4. Teste seu plug-in no RRPG Firecast.

 

Observações:

  • O comando “rdk i” não instala o plug-in caso encontre algum erro no projeto.
  • Se o RRPG Firecast estiver em execução, o comando “rdk i” automaticamente descarrega o plug-in, instala a nova versão e recarrega o plug-in.
  • Se o RRPG Firecast estiver fechado, o comando “rdk i” realiza uma instalação “off-line” e a nova versão será carregada quando o RRPG Firecast executar.
