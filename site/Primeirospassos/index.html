Primeiros passos

 

Agora que você já instalou um bom editor de texto, você já pode criar seu plug-in. Siga estas orientações:

 

  1. Certifique-se se você já baixou e instalou o Firecast SDK 3, pois, sem ele, este passo-a-passo não funcionará. Você pode baixá-lo em https://firecast.app/para-programadores/\  
  2. Crie um novo projeto de plug-in. Saiba como em Criando um novo projeto de plug-in.

 

  1. Adicione funcionalidades ao seu projeto de plug-in:
  2. Deseja fazer um modelo de ficha? Veja Modelos de Fichas.
  3. Deseja converter um PDF para um modelo de ficha? Veja o tutorial: Criar um modelo de ficha à partir de um PDF
  4. Veja também Tutoriais para ter mais ideias =).

 

  1. Teste seu plug-in! Saiba como em Testando o projeto de plug-in.

 

  1. Distribua seu plug-in
