# Projetos de Plug-in

&nbsp;

Um **projeto de plug-in** é um **diretório** no computador que contém todo o código fonte e arquivos auxiliares que são compilados e empacotados sob a forma de um único arquivo de extensão ‘.rpk’, o **plug-in**.
